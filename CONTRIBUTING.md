## Terms of Contributing
All developers are welcome to contribute to this project, I just ask that you follow the guidelines below:
- Any new features should be brought up in issues first
- When making changes to the project, do not commit to the master branch, fork the project and make changes on your personal fork, then make a pull request
- Remember to add your name to the list of contributors!
